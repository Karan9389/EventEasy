## 📖 About The Project

**EventEase** is a modern, responsive event planning and vendor booking platform. Designed to take the stress out of event curation, it connects users with elite event professionals—from breathtaking photography and live music to exquisite catering and professional makeup artistry.

Built with performance and user experience in mind, the application features smooth animations, state management for booking, and a sleek, glass-morphism inspired UI.

### 🚀 Key Features

* **Categorized Vendor Discovery:** Seamlessly browse through elite Photographers, Caterers, Musicians, and Makeup Artists.
* **Smart Cart & Booking System:** Add services to your cart, view estimated taxes, and manage your total event budget in real-time.
* **Personal Wishlist:** Save your favorite vendors to your personal wishlist for quick access later.
* **User Dashboard:** Dedicated profile section to track upcoming events and monitor recent booking activity.
* **Fluid UI/UX:** Enhanced with `framer-motion` for buttery-smooth page transitions and element reveals.

* **Core:** [React 19](https://react.dev/) & [TypeScript](https://www.typescriptlang.org/)
* **Build Tool:** [Vite](https://vitejs.dev/)
* **Styling:** [Tailwind CSS v4](https://tailwindcss.com/)
* **Routing:** [React Router v7](https://reactrouter.com/)
* **Icons & Animation:** [Lucide React](https://lucide.dev/) & [Motion (Framer)](https://motion.dev/)
